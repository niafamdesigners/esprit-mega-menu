نمایش منوی پویا

این صفحه نشان می‌دهد که چگونه می‌توانید منو را از داده‌های JSON به صورت پویا تولید کنید.

ویژگی‌های منوی پویا:

  • بارگذاری منو از JSON
  • به‌روزرسانی آیتم‌های منو
  • اضافه/حذف آیتم‌های منو
  • صادرات تنظیمات فعلی
  • اعتبارسنجی ساختار داده

نحوه استفاده در پروژه شما:

  1. داده‌های منو را در قالب JSON آماده کنید
  2. کلاس EspritDynamicMenu را import کنید
  3. نمونه‌ای از کلاس ایجاد کنید
  4. داده‌ها را بارگذاری و منو را تولید کنید

ساختار JSON مورد نیاز:

{
  "menuData": {
    "logo": {
      "text": "نام لوگو",
      "url": "#"
    },
    "items": [
      {
        "id": "unique-id",
        "title": "عنوان منو",
        "url": "#",
        "type": "simple|mega|dropdown",
        "columns": [...], // برای نوع mega
        "items": [...] // برای نوع dropdown
      }
    ]
  }
}
